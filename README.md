## Sales Forecasting and Inventory Planning Dashboard

This project is an end-to-end Data Science and Analytics pipeline built using Python, Pandas, Scikit-learn, Streamlit, and Plotly. The system cleans and processes raw sales data, generates KPIs and business insights, forecasts future product demand and revenue using Linear Regression, and recommends inventory ordering quantities based on projected demand and safety stock logic. The final outputs are visualized through an interactive Streamlit dashboard containing revenue analytics, forecasting graphs, top and worst selling products, and inventory recommendations.

## Features

* Data cleaning and preprocessing
* KPI generation and sales analytics
* Revenue forecasting using Linear Regression
* Product demand prediction
* Inventory recommendation system
* Interactive Streamlit dashboard

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Plotly
* Streamlit

## Run the Project

Install dependencies:
pip install -r requirements.txt

## Run the dashboard:
cd app
python -m streamlit run app.py
